# 777-Utilities <img align="right" src="https://jitpack.io/v/777Code-pl/777-commands.svg" alt="JitPack Badge" />
Simple bukkit utilities library

### Repository
For maven:
```xml
<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>
```

For gradle:
```groovy
repositories {
    maven { url 'https://jitpack.io' }
}
```

### Dependency

For maven:
```xml
<dependencies>
    <dependency>
        <groupId>com.github.777Code-pl</groupId>
        <artifactId>777-commands</artifactId>
        <version>1.0.0</version>
    </dependency>
  </dependencies>
```

For gradle:
```groovy
dependencies {
    implementation 'com.github.777Code-pl:777-commands:v1.0.0'
}
```
